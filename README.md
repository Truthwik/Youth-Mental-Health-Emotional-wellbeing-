# Svasthya Company About Us Page

This is the official "About Us" page for **Svasthya**, a health and wellness company.

## Features:
- **Project Concept**: Svasthya (Health & Wellbeing).
- **Specialist Team Section**: featuring 4 team members with their specialty, experience, and joining year.
- **Micro-Animations**: Cards lift on hover and slide in as you scroll.
- **Animated Avatars**: Custom-generated minimalist flat-illustration avatars for each member.
- **Modern Typography**: Powered by **Outfit** Google Font.

## Structure:
- `index.html`: Main page layout.
- `style.css`: Modern Wellness UI design system.
- `assets/`: Team avatars.

To view: Open `index.html` or run the local server.
